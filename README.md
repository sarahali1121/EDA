🦠 COVID-19 India — EDA & Recovery Prediction
Exploratory Data Analysis on India's state-wise COVID-19 data with a Logistic Regression model to predict recovery probability.

📊 What This Project Does

Loads and explores COVID-19 data across Indian states
Calculates total confirmed, cured, death, and active cases
Visualizes case distribution by state using bar charts
Trains a Logistic Regression model to predict recovery probability
